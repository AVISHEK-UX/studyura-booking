## Layers

| Layer | Technology | Responsibility |
|-------|-----------|----------------|
| Presentation | React, Tailwind, shadcn/ui | UI rendering, routing |
| State | React Query | Server state caching |
| Backend | Lovable Cloud (Supabase) | DB, Auth, Storage |
| CDN | Auto (Lovable hosting) | Static assets delivery |

## Security Model
- **Public routes**: Read-only access to active libraries
- **Admin routes**: Protected by authentication
- **RLS Policies**: Enforce read/write at database level
- **Storage**: Public read for library images, authenticated write
